# Idea Brief — Adaptive Memory Retrieval for Long-Horizon Agents

## Problem
Current long-horizon agents often use static retrieval triggers even though uncertainty changes during execution.

## Proposed approach
Use uncertainty-guided retrieval scheduling so the agent retrieves memory only when reasoning confidence drops below a learned threshold.
